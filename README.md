## 项目简介 Description

**http-server** 基于**Boost.Asio Boost.Beast** 进行编写，提供异步的http服务器实现

- 设计背景：学习Asio异步模型，异步服务器的工作原理

## 包基础

    - **Asio**
    - **Beast**

## 功能特性 Features

> 实现 HTTP/1.1 异步并发 超市控制
